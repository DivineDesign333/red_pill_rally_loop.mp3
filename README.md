# red_pill_rally_loop.mp3

## Configuration

This project supports the following configuration options:

- `USE_VULKAN=true` - Enable Vulkan graphics API support
- `USE_AIO=true` - Enable Async I/O support

Configuration can be set via:
- Environment variables (`.env` file)
- Configuration script (`config.sh`)

To apply the configuration, source the config file:
```bash
source config.sh
```
