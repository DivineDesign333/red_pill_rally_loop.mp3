# Dashboard

This directory contains the deployment dashboard application.

## Files

- `index.html` - Main dashboard page with embedded styles and scripts
- `api.js` - API module for fetching deployment data from GitHub
- `styles.css` - Additional styles for animations and responsive design

## Features

- Real-time deployment status monitoring
- Deployment history tracking
- Statistics and metrics display
- Responsive design for mobile and desktop
- Dark mode support
- Accessibility features

## Development

To test locally:
1. Open `index.html` in a web browser
2. The dashboard will display sample data

## Production

The dashboard is automatically deployed to GitHub Pages via the Deploy Dashboard workflow.

## Customization

You can customize the dashboard by:
- Modifying colors in the CSS
- Updating the API endpoints in `api.js`
- Adding new statistics or metrics
- Integrating with additional monitoring tools
